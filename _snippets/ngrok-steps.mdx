1. Install ngrok:

<CodeGroup>
```sh Mac
brew install ngrok/ngrok/ngrok
```

```sh Windows
choco install ngrok
```

</CodeGroup>

2. Open a new terminal window/tab, you need to leave this running

3. Create an http tunnel at port 3000:

```sh
ngrok http 3000
```

4. Grab your forwarding address in the ngrok output:

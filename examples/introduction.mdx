We have an [examples repo](https://github.com/triggerdotdev/examples) that we add to all the time. It's a great place to start if you're looking for inspiration.

Alternatively you can jump directly to an example using the table below:

| Name                                                                       | Description                                                                                                          | Integrations |
| -------------------------------------------------------------------------- | -------------------------------------------------------------------------------------------------------------------- | ------------ |
| [Delays](https://github.com/triggerdotdev/examples/tree/main/delays)       | Using delays inside Jobs                                                                                             | N/A          |
| [GitHub](https://github.com/triggerdotdev/examples/tree/main/github)       | When a new GitHub issue is opened it adds a "Bug" label to it. When a repo is starred it tells you the total number. | GitHub       |
| [OpenAI](https://github.com/triggerdotdev/examples/tree/main/openai)       | Generate images and jokes from a prompt                                                                              | OpenAI       |
| [Resend](https://github.com/triggerdotdev/examples/tree/main/resend)       | Sends an email by submitting a form in the Next.js app                                                               | Resend       |
| [Scheduled](https://github.com/triggerdotdev/examples/tree/main/scheduled) | Interval and cron scheduled Jobs                                                                                     | N/A          |
| [Slack](https://github.com/triggerdotdev/examples/tree/main/slack)         | Sends a Slack message when an event is received                                                                      | Slack        |
